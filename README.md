# Pencilon — Advertising Agency Website

A premium, production-ready agency website for **Pencilon** — a full-service advertising agency based in Malappuram, Kerala specialising in Branding, Digital Marketing and Social Media Creatives.

## 🚀 Live Deploy

Deploy instantly to **GitHub Pages**:
1. Push this repo to GitHub
2. Go to **Settings → Pages → Source → main / root**
3. Your site goes live at `https://yourusername.github.io/pencilon`

## 📁 Project Structure

```
pencilon/
├── index.html          ← Main HTML
├── css/
│   └── style.css       ← All styles + animations
├── js/
│   └── main.js         ← Interactions, cursor, scroll effects
├── assets/
│   └── logo.png        ← Pencilon logo
└── README.md
```

## ✨ Features

- Custom animated cursor
- Scroll-triggered reveal animations
- Sticky transparent nav with blur on scroll
- Horizontal ticker / marquee
- Animated hero with large background letterform
- Stats counter animation
- Service cards with hover effects
- Work portfolio grid
- Process accordion list
- Testimonial section
- Full responsive mobile layout
- Zero dependencies — pure HTML/CSS/JS
